# Giggle-bot
## The bot to spice up your slack room

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

### How to contribute
- fork repository
- submit PR to `development` branch

### How to run
 - requires node.js
 - go to root folder
 - run  `npm install`
 - run  `SLACK_TOKEN=<token_from_slack> node index.js`
